# Student-Management-System
The Student Management System is a console-based Java application that allows users to add, view, update, and delete student records efficiently. It uses object-oriented principles and ArrayList for in-memory data storage, providing a simple, menu-driven interface for managing student information.
# Student Management System in Java

## Overview
This is a simple console-based Student Management System implemented in Java. It allows users to manage student records by performing core CRUD operations: adding new students, viewing all students, updating existing student details, and deleting student records.

## Features
- Add student with ID, name, and email
- View all student records
- Update student information by ID
- Delete student records by ID
- Simple, menu-driven interface using the console

## Technologies Used
- Java SE (Standard Edition)
- Java Collections framework (ArrayList)
- Basic console input/output using Scanner
